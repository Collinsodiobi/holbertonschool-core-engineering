## Local Model Setup

This project uses Ollama as the default model provider.

Environment variables:

MODEL_PROVIDER=ollama
MODEL_NAME=llama3.2
OLLAMA_URL=http://localhost:11434

Install Ollama on your local machine and run:

ollama pull llama3.2
